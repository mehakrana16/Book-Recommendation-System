# 📚 Book Recommendation System

This project is a **Book Recommendation System** designed to help users discover books they might enjoy based on their preferences or popular trends. The system uses **machine learning techniques** like content-based filtering and collaborative filtering to generate personalized recommendations.

---

## 🔍 Objective
The goal of this project is to **recommend books intelligently** by analyzing user behavior, preferences, and book features such as **title, author, and genre**.

---

## 💡 Key Features

- **Collaborative Filtering**: Recommends books based on user ratings and behavior patterns.  
- **Popular Books Section**: Highlights top-rated and most-read books.  
- **Search Functionality**: Allows users to search for any book title in the dataset.  

---

## 🛠️ Technologies Used

- **Python**: Core programming language for logic and model building.  
- **Pandas & NumPy**: For data manipulation and preprocessing.  
- **Scikit-learn**: Implements similarity algorithms like cosine similarity and nearest neighbors.  
- **Jupyter Notebook**: Used for data cleaning, preprocessing, and testing models.  
- **Flask**: Deploys the recommendation system with a simple web interface.  

---

## 📊 Dataset
The dataset used for this project is available on Kaggle:  
[Book Recommendation Dataset](https://www.kaggle.com/datasets/arashnic/book-recommendation-dataset)  

- **Total entries:** 271,361 rows  
- **Features include:** User IDs, Book titles, Authors, Ratings
